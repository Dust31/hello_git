1. Винтажный калькулятор с интерфейсом

Старинный металлический калькулятор 1970-х годов

Создайте калькулятор с интерфейсом, вдохновленным винтажными механическими калькуляторами середины XX века.

Особенности проекта:

  • Интерфейс с металлическими кнопками и механическими переключателями
  • Анимации нажатия клавиш с тактильной отдачей
  • Звуки механических операций при вычислениях
  • Эффект сепии и винтажные шрифты для отображения результатов
  • Возможность распечатать результат на стилизованной "бумажной" ленте

Технологии: HTML, CSS, JavaScript, возможно использование Canvas для анимаций.

2. Телеграм бот с историческими фотографиями

Классический фотоаппарат для архивных снимков

Разработайте телеграм-бота, который отправляет пользователям исторические фотографии из разных эпох.

Особенности проекта:

  • База данных с классифицированными историческими фотографиями
  • Возможность запроса фотографий по темам (например, "мода 1920-х", "советская техника")
  • Информация об историческом контексте каждой фотографии
  • Система рейтинга и комментариев пользователей
  • Режим "день в истории" с фотографией соответствующего дня

Технологии: Python, API Telegram, база данных SQL или MongoDB, обработка изображений.

3. Винтажный макрос для автоматизации

Классическая пишущая машинка - прообраз макросов

Создайте систему макросов, которая имитирует работу старых механических устройств для автоматизации задач.

Особенности проекта:

  • Интерфейс записи и воспроизведения макросов
  • Стилизация под старые механические устройства (пишущие машинки, счетные машины)
  • Библиотека готовых макросов для стандартных задач
  • Система планирования выполнения макросов
  • Визуализация процесса выполнения с механическими анимациями

Технологии: Python, возможно расширения для Office или браузеров, GUI-интерфейс (Tkinter или PyQt).
